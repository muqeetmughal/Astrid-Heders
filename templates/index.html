A Quick introduction to Astrid approach to Hedera.



Fetch account balance

With this function we are able to retreive the amount of tokens held by a given account on Hedera testnet. Once set up the account on testnet with the simplified process of Hedera Portal, we have the gateway for the chain. Given the operator keys, we are able to connect to the chain through Hedera SDK, take the Account ID as input of the form, and use it to call the 'account balance query' function. The output shows the amount of HBAR tokens held.



Store a file into Hedera Blockchain

This function uses Hedera SDK to create and store a new file (a string) into Hedera Testnet. Once we setup the Operator Keys, we are able to execute a specific transaction to store data called 'File Create Transaction'. Insert the text you want to save on testnet above, and then let's enjoy the transaction ID and the File content stored.



Generate new account

Hedera SDK have a great options to create new wallets in a safe manner. There are native package functions that are able to safely generate private and public keys, and then specific transactions that are able to safely and fastly create a new account on testnet. Clck the button below and let's find out how it fast and easy is. In the last button replace 'Generate new wallet' with 'Generate new account'
